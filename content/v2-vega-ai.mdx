# Vega AI
AI-Powered Document Processing Platform

Built an intelligent automation platform that transforms raw documents into structured data using Google Gemini, Claude, and Fireworks AI. The system processes multiple file formats in parallel and delivers validated outputs through a dynamic schema engine.

- Tech Stack: Python, Google Gemini API, AWS Bedrock, AWS S3, CloudFront
- Key Results: Automated end-to-end content pipeline with 100% output validation
- Features: Batch processing, OCR automation, cloud deployment
