# Compound Interest Calculator
High-Accuracy Financial Prediction Model

Developed a mathematical model for rapid compound interest approximation without calculators. The piecewise quadratic approach replaces complex exponential formulas with simple arithmetic, validated across 60 test scenarios.

- Tech Stack: Python, NumPy, Pandas, Matplotlib
- Key Results: 0.74% average error rate across 3-15% interest rates
- Features: Manual calculation optimization, comprehensive error analysis
