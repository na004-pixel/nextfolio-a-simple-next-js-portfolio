# Automated S3 Backup System
Git-Aware Cloud Backup Solution

Created a robust backup pipeline that intelligently snapshots directories to AWS S3. The system respects .gitignore rules, uses rsync for efficiency, and provides comprehensive logging with safety checks.

- Tech Stack: Bash, AWS S3, Git, rsync
- Key Results: Automated disaster recovery with versioned snapshots
- Features: Incremental backups, parallel uploads, audit trails
